Droit Canadien sur les Jeux de Hasard

Le Code criminel du Canada a été modifié en 2021 (Projet de loi C-218) pour permettre les paris sportifs sur un seul match. Avant cela, seuls les paris combinés étaient légaux via les sociétés de loterie provinciales.

Ontario a été le plus loin : iGaming Ontario a été lancé en avril 2022 et compte désormais plus de 50 opérateurs agréés, y compris DraftKings, BetMGM, Bet365, PointsBet, FanDuel, et plus encore. C'est le marché réglementé le plus ouvert en Amérique du Nord.

D'autres provinces (CB, Alberta, Québec, Manitoba, etc.) fonctionnent encore via des sociétés de la Couronne provinciale (PlayNow, Sports Interaction via Loto-Québec, etc.), mais les résidents peuvent également accéder légalement à des sites offshore.

Taux d'Imposition sur les Paris Canadiens

Le Canada n'a aucun impôt fédéral sur les gains de jeu pour les parieurs récréatifs. Les joueurs professionnels peuvent être soumis à l'impôt sur le revenu. La position de l'ARC : si le jeu est votre principale source de revenu, les gains sont imposables. Pour les parieurs récréatifs, gardez tout.
